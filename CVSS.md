# CVSS

Schadenspotenzial als Zahl einer [[Vulnerability]]

## Exploitability

- Attack Vector (AV)
- Attack complexity (AC)
- Privileges required (PR)
- User interaction (UI)

## Impact

- Scope Changed (AC)
- [[Confidentiality]] (C)
- [[Integrity]] (I)
- [[Availability]] (A)

## Severity

- None (0.0)
- Low (0.1 - 3.9)
- Medium (4.0 - 6.9)
- High (7.0 - 8.9)
- Critical (9.0 - 10.0)
