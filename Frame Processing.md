# Frame Processing

Geräte nehmen nur [[Frame]]s an, die an sie adressiert werden, also wenn die Destination [[MAC-Adresse]] mit der Gerät-[[MAC-Adresse]] übereinstimmt. Sonst werden die [[Frame]]s ignoriert.
