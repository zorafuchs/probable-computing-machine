# MAC Fludding

Überfüllung der [[Switch]]-Tabelle, bei welchem Port welche [[MAC-Adresse]] hingehört, damit der [[Switch]] alle [[Frame]]s an alle [[Port]]s schickt. Dadurch kann eine Verbindung abgehört werden.
