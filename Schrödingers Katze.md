# Schrödingers Katze

1. Katze in einer zugemachten Box
2. Es wird mit 50% giftiges Gas eingeströmt
3. Man weiss bis zum öffnen nicht, ob die Katze tot oder lebendig ist

Das System kollabiert sobald man die Katze sieht
